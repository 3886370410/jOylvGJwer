# 项目简介

## 核心功能
- 管理员角色：提供用户管理、商家管理、商品分类管理、宠物商品管理、售后服务管理、通知公告管理、留言投诉管理、系统设置、订单处理和个人中心等功能。
- 商家角色：上传商品、管理订单、查看销售数据等。
- 用户角色：浏览商品、下单购买、查看订单状态、咨询在线客服等。
- 宠物商品推荐：采用协同过滤算法进行商品推荐。
- 在线咨询：提供用户与商家之间的即时沟通功能。

## 适用场景
- 宠物用品零售商
- 宠物用品电商平台
- 宠物爱好者社区

## 技术特点
- 开发语言：Java
- 框架：SpringBoot、Vue
- 数据库：Mysql
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Maven、Navcat
- 部署：支持远程部署和答疑解惑服务
